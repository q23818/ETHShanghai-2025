# [Project] w3hub: Web3资产管理工具

## 项目概述
w3hub是一个帮助Web3用户智能管理资产的安全工具，解决以下痛点：
- 🛡️ 资产安全监控：实时追踪多链资产变动
- 🔔 智能预警：价格暴跌、可疑交易即时通知
- 📊 统一视图：跨链资产一站式管理

## 技术实现

### 核心架构
```mermaid
graph TD
    A[CLI/Web界面] --> B[REST API]
    B --> C[资产管理模块]
    C --> D[以太坊适配器]
    C --> E[SQLite数据库]
```

### 关键技术
- 多链支持架构（已实现以太坊）
- 可扩展的预警规则引擎
- 双通道通知（邮件+Telegram）
- Swagger API文档集成

## 功能亮点
1. **实时监控**：
   - 跟踪地址资产变动
   - 交易历史查询
2. **安全预警**：
   - 大额交易提醒
   - 价格波动监控（1小时跌幅>10%触发）
3. **数据持久化**：
   - 资产快照存储
   - 历史记录查询

## 使用说明
```bash
# 启动服务
go run cmd/w3hub/main.go

# API文档
http://localhost:8080/docs/
```
